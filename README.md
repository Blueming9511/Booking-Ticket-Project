## Công Nghệ Sử Dụng
- Backend: Java Spring Boot
- Frontend: React.js
- Database: MySQL
- Payment Gateway: VNPAY

## Cài Đặt và Chạy Dự Án

### Backend
1. Cài đặt Java JDK 17
2. Cài đặt Maven
3. Chạy lệnh:
```bash
cd backend
mvn spring-boot:run
```

### Frontend
1. Cài đặt Node.js
2. Chạy lệnh:
```bash
cd frontend
npm install
npm start
```
